## 基本操作步骤
编译运行，输入要猜的残缺单词，未知部分用\*表示。

例如，ap\*le

按下回车。

接下来输入你已经猜过但是猜错了的字母，中间不需要打空格，回车结束。

例如你猜了但是猜错了的字母有b,c,d

那么你就输入bcd和一个回车

接下来程序就会给你所有可能性以及其中出现每个字母的次数，方便你选择接下来要猜什么。
